Economy Of Hours

Templates para WordPress

Foram desenvolvidos templates para criar páginas dentro do WordPress, seguindo o design responsivo do layout existente. Estes templates possuem todos os seus elementos editáveis, podendo o cliente alterar títulos, textos e conteúdo como precisar, de forma simples e didática.

Desenvolvido em 2016 para Marcelo Graciolli.

economyofhours.com

Editor do template

Dentro do WordPress os templates receberam atenção no painel onde as informações são inseridas, de forma a ter uma experiência de edição intuitiva e simples.
