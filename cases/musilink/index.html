Musilink

Loja virtual WooCommerce

Neste projeto foi elaborado um tema para a plataforma de loja virtual WooCommerce, buscando criar uma experiência mais eficiente para os compradores, focando-se no conteúdo informativo de cada produto e na forma de navegação pela loja até a compra do produto.

Desenvolvido em 2018 para Marcelo Graciolli.

www.musilink.com.br

Compras pelo celular

Lojas virtuais costumam ter uma experiência ruim de uso em dispositivos móveis, principalmente no processo de compra de um produto. Aqui o layout e sua navegação foi pensada para tornar o uso fácil e permitir uma resposta adequada ao cliente que esta procurando comprar um produto.
